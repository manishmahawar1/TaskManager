# task manager app

This is a simple task manager application where users can add, edit, delete and mark tasks as completed.

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed


## How to Run

1. Clone the repo
2. Install dependencies:
   npm install

3. Start backend:
   npm start

4. Start frontend:
   npm run dev


## Tech Stack

- React
- Node.js
- Express



## Assumptions

- This project is made for a single user
- No authentication is implemented
- Focus is on basic CRUD operations
